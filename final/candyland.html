Candyland Game

A Classic Board Game Built in C++

About the Game

Candyland is a multiplayer board game where players compete to reach the Candy Castle first while solving puzzles and using candies to strategically slow down opponents.

Project Goal

The goal of this project was to recreate the classic Candyland board game as an interactive C++ program. I focused on building clear turn-based logic, using text files to drive the game setup, and creating a fun, replayable experience for multiple players.

My Contributions

  • Designed the game flow, including turn order, win conditions, and rules for special tiles.
  • Implemented the core game logic in C++, including player movement, card drawing, and score tracking.
  • Used external text files to store game messages and board information so the game could be updated without changing code.
  • Tested gameplay, fixed logic bugs, and refined prompts to make the game easy to understand for new players.

Skills Gained

  • C++ programming (functions, structs/classes, control flow, and loops)
  • File input/output and working with external text resources
  • Problem-solving and debugging game logic
  • Documenting a project for other users with a README and clear instructions

How to Play

  1. Download the game by clicking the link below.
  2. Unzip the downloaded file to a folder.
  3. Ensure the text_files/ folder and candyland.out file are in the same directory.
  4. Run the game with the following command in a terminal: 
    ./candyland.out

For more detailed instructions, refer to the README file.
