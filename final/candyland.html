Candyland Game

A Classic Board Game Built in C++

About the Game

This project implements an interactive board game fashioned after the game Candyland. It is a multiplayer game where users compete to reach the Candy Castle first while solving puzzles and using their wit to slow opponents down with candies.

How to Compile and Run

Compile: g++ -std=c++17 boardDriver.cpp candystore.cpp player.cpp games.cpp board.cpp menu.cpp tile.cpp -o candyland.out
Run: ./candyland.out

Dependencies:
candies.txt, characters.txt, riddles.txt must be in the ./text_files/ directory.
Ensure these headers are included:
- board.h, games.h, menu.h, player.h, tile.h, candystore.h

For more details, download the README file.
