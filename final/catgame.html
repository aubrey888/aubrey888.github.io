Cat Game

A Fun Adventure Game Built in Python

About the Game

This project is a Python-based adventure game where players guide a mischievous cat through various challenges. Use your wits to overcome obstacles and collect treasures!

Project Goal

The goal of this project was to create an engaging, text-based adventure game using Python. I wanted to explore game design fundamentals, user decision-making logic, and interactive storytelling while practicing clean, modular code structure.

My Contributions

  • Designed the storyline, character interactions, and decision paths for the adventure.
  • Developed core gameplay mechanics including movement, inventory collection, and challenge obstacles.
  • Implemented Python functions to structure game events and manage player choices.
  • Debugged edge cases to ensure smooth gameplay and logical progression.

Skills Gained

  • Python programming (functions, variables, loops, conditionals)
  • Building interactive, decision-driven games
  • Logical problem solving and handling user input
  • Structuring projects with multiple files and assets

How to Play

  1. Download the game by clicking the link below.
  2. Unzip the file into a folder.
  3. Run the game with the following command in a terminal: 
    python3 main.py

For more details, look to the README file.
