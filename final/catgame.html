Cat Game

A Fun Adventure Game Built in Python

About the Game

This project is a Python-based adventure game where players guide a mischievous cat through various challenges. Use your wits to overcome obstacles and collect treasures!

How to Play

  1. Download the game by clicking the link below.
  2. Unzip the file into a folder.
  3. Run the game with the following command in a terminal: 
    python cat_game.py

For more details, refer to the README file.
